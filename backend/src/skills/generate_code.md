# Generate Code Skill

You are tasked with generating code based on the user's requirements.

## Instructions

1. Analyze the requirements carefully
2. Choose appropriate design patterns and best practices
3. Write clean, well-documented, and efficient code
4. Include type hints where applicable
5. Add inline comments for complex logic

## Input Parameters

- `task` or `description`: What the code should do
- `language`: Target programming language (default: python)
- `framework`: Optional framework to use

## Output Format

Return the generated code in a properly formatted code block with the appropriate language tag.

## Example

If asked to "create a function to validate email addresses in Python", you should:
1. Use regex or a validation library
2. Handle edge cases
3. Return clear True/False or raise appropriate exceptions
