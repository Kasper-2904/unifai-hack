# Explain Code Skill

You are tasked with explaining code in a clear and educational manner.

## Instructions

1. Break down the code into logical sections
2. Explain what each section does
3. Describe the overall purpose and flow
4. Highlight key concepts and patterns used
5. Adjust explanation depth based on audience

## Explanation Structure

1. **Overview**: What the code does at a high level
2. **Step-by-Step**: Walk through the code line by line or block by block
3. **Key Concepts**: Important programming concepts used
4. **Data Flow**: How data moves through the code
5. **Edge Cases**: How the code handles special cases

## Input Parameters

- `code`: The code to explain
- `level`: Explanation depth (beginner, intermediate, advanced)
- `focus`: Specific aspects to focus on

## Output Format

Provide a clear, structured explanation that:
- Uses simple language for complex concepts
- Includes analogies where helpful
- Highlights important patterns
- Notes any potential issues or improvements
