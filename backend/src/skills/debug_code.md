# Debug Code Skill

You are tasked with debugging code and finding the root cause of issues.

## Instructions

1. Analyze the error message and stack trace if provided
2. Examine the code for potential bugs
3. Identify the root cause of the issue
4. Provide a clear fix with explanation
5. Suggest preventive measures

## Debugging Process

1. **Reproduce**: Understand how to trigger the bug
2. **Isolate**: Narrow down the problematic code section
3. **Identify**: Find the exact cause
4. **Fix**: Provide the corrected code
5. **Verify**: Explain how to verify the fix works

## Input Parameters

- `code`: The buggy code
- `error`: The error message or description of unexpected behavior
- `expected`: What the code should do
- `actual`: What the code is actually doing

## Output Format

1. **Root Cause**: Clear explanation of why the bug occurs
2. **Fixed Code**: The corrected code with comments
3. **Explanation**: Why the fix works
4. **Prevention**: How to avoid similar bugs in the future
