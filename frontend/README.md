# Frontend

React app for developer, PM, marketplace, and billing views.

## Scope
- Developer dashboard
- PM dashboard
- Agent marketplace screens
- Billing/subscription views

## Run
```bash
cd frontend
npm ci
npm run dev
```

## Quality
```bash
cd frontend
npm run lint
npm run build
```

## Tests
- Place tests under `src/**/*.test.ts(x)`.
- `src/__tests__/` is tracked for shared test scaffolding.

See root `AGENTS.md` and `TESTING.md` for shared team rules.
